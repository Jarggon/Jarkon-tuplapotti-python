How to Play (Windows)  
1. Download the latest .zip file from the Releases page. *DO NOT RUN* the game directly from inside the ZIP file!  
2. Right-click the downloaded .zip file and select Extract All...  
3. Open the newly extracted folder.  
4. Double-click the Tuplapotti_2.0.exe file.  
(Note: Because this is a new game made by an independent developer, Windows SmartScreen might show a "Windows protected your PC" warning. You can safely bypass this by clicking "More info" and then "Run anyway".)

Your balance will automatically save to the "jarkon settings" folder located right next to the .exe file.

Tip for Desktop Shortcut: If you want a quick way to launch the game from your desktop, do NOT move the .exe file itself! Instead, right-click Tuplapotti_2.0.exe and select: Show more options -> Send to -> Desktop (create shortcut).
